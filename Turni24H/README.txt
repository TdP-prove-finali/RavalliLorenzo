Turni 24H, software per la gestione di turni

Istruzioni per l'installazione e l'utilizzo:

1- Effettuare il fork del progetto (URI : https://github.com/TdP-prove-finali/RavalliLorenzo);

2- Importare il progetto nell'IDE Eclipse;

3- Caricare il data-set ('turni.sql') presente nell'apposita cartella ('database') sul proprio database SQL;

4- Guardare il video illustrativo del funzionamento dell'applicazione al seguente link: https://youtu.be/2AwoMV82ah8

5- Eseguire la classe 'Main' per lanciare il programma;
